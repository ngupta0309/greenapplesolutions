Product Features

XipTM Micropayment System has been designed around established communication standards with focus on the ease of integration with existing mobile wallets, electronic banking accounts and telecom networks. It is built for high scalability, high security and high availability.

XipPOS

Mobile POS for merchants

  • Quad-based GSM/ GPRS
  • Credit card size
  • Ruggedized design for use in harsh environments
  • Bright color TFT display
  • Long battery life
  • Contactless NFC interface
  • Ergonomic numeric keypad
  • Shortcut keys for frequent operations
  • Hook for neck strap/ belt clip
  • GPRS with USSD fallback connectivity
  • XipAGENT - specially designed for provisioning of XipPOS and XipTAG at MNO

XipTAG

Contactless card for consumers

  • High security card
  • Long life
  • Space for consumer personalization
  • Laser etched ID for visual identification
  • Designed for co-branding with the MMSP
  • Option for sticker versions

XipMS

XipTM mediation and web server

  • COTS Intel-Linux rack mount servers
  • Network glue elements
  • High availability through N+1 redundancy support
  • On-net system with secured interface to the mobile money platform and other network elements
  • Administration interface
  • Integration with customer care system
  • Interface for NOC, SNMP support
  • Remote administration and support

System Highlights

  • GPRS, USSD and SMS connectivity
  • Supports wide range of mobile money and banking systems
  • Secured tunnel for consumer PIN Information
  • Merchant XipPOS and consumer XipTAG provisioning
  • Ticker for promotions
  • Fraud prevention
  • Speeds up cash-in / cash-out and airtime sales

System Features

  • Connects directly to the merchant's mobile money account and can be configured for a 'receive only' mode, enabling larger merchants to issue a XipPOS to each employee
  • Its sophisticated contactless technology and small size allows easy transactions in all kinds of practical situations that merchants face in their daily life
  • XipTAG can be paired with mobile network from any MNO. Only the mobile money account ID needs to be from the same service.
